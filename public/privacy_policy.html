ProcessSpy macOS App Privacy Policy

The ProcessSpy macOS app respects your data and privacy. We collect minimal data that is required for the app to work: info about running processes and their open files and sockets

Data we do NOT collect

  • The app does not collect any kind of analytical data, not even anonymous analytics or crash log.
  • The app does not collect any kind of personal data.
  • No internet connection required to use the app. The only features that need internet connection is auto update and search process name online.
  • The app does not make any changes to the system without explicit permissions from you.
  • The app may ask for permissions in the following scenarios:
    • 'Show in Finder' in the main process table is used
    • Context menu in Files tab is opened
    • Process file is opened using 'Open with' context menu command
    • Feature 'Find process with mouse click' is used
